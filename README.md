# 🕌 Website Donasi Online Renovasi Masjid Al-Furqon

## 📖 Deskripsi Proyek

Website Donasi Online Renovasi Masjid Al-Furqon merupakan proyek yang dirancang untuk membantu proses penggalangan dana renovasi masjid secara transparan, mudah diakses, dan dapat menjangkau donatur dari berbagai daerah.

Proyek ini dibuat sebagai studi kasus dalam mata kuliah **Manajemen Proyek Sistem Informasi** Universitas Esa Unggul. Sistem dikembangkan menggunakan platform WordPress dan plugin donasi untuk mempermudah pengelolaan donasi tanpa membangun aplikasi dari nol.

Melalui website ini, masyarakat dapat melihat informasi renovasi, melakukan donasi, memantau perkembangan renovasi, serta mengakses laporan donasi secara transparan.

---

## 🎯 Tujuan Proyek

* Memudahkan masyarakat melakukan donasi kapan saja dan dari mana saja.
* Meningkatkan transparansi pengelolaan dana renovasi masjid.
* Membantu DKM dalam pencatatan dan pelaporan donasi secara digital.
* Mempercepat proses pengumpulan dana renovasi Masjid Al-Furqon.

## 📋 Ruang Lingkup Sistem

### In Scope

* Landing page profil masjid.
* Form donasi online.
* Halaman progres renovasi.
* Dashboard admin DKM.
* Laporan donasi publik.

### Out of Scope

* Aplikasi mobile native.
* Payment gateway otomatis.
* Donasi berulang (recurring donation).
* Multi campaign donation.

## 📊 Metodologi Pengembangan

Metodologi yang digunakan adalah **Agile Scrum (Solo Scrum)** karena proyek dikerjakan secara individu dengan pendekatan sprint yang fleksibel dan memungkinkan evaluasi berkala bersama stakeholder.

Tahapan utama:

1. Inisiasi Proyek
2. Analisis Kebutuhan
3. Perencanaan

5. Pengujian
6. Deployment
7. Monitoring dan Evaluasi
